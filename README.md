## OpenAction Discord plugin

An OpenAction ([OpenDeck](https://github.com/nekename/OpenDeck) / [Tacto](https://tacto.rivul.us)) plugin for controlling the Discord desktop client

> [!WARNING]
> This plugin is currently in an alpha state, with many planned actions not yet implemented. If you need more functionality, install the proprietary Discord plugin developed by Elgato from the Elgato App Store archive in the OpenDeck plugin manager (or use the [direct download](https://plugins.amankhanna.me/rezipped/com.elgato.discord.zip)).

#### Actions

- Toggle Mute
- Toggle Deafen
- Push to Mute
- Push to Talk
